# 🚀 Giga Chad Wallet

A modern Solana-based trading platform that combines embedded wallets, real-time market data, portfolio tracking, and simulated trading into a seamless web experience.

The application integrates Privy authentication, Supabase, BirdEye APIs, and Solana Devnet to deliver an intuitive crypto trading interface while maintaining a scalable frontend architecture.

---

## 🌐 Live Demo

👉 https://giga-chad-wallet.vercel.app

---

## ✨ Features

- 🔐 Google & Email Authentication (Privy)
- 👛 Embedded Solana Wallets
- 📈 Real-Time Market Data
- 💰 Simulated Buy & Sell Orders
- 📊 Portfolio Tracking
- ⭐ Watchlist Management
- 🏆 Leaderboard
- 📉 Interactive OHLCV Charts
- 👥 Holder Statistics
- 🔄 Automatic Fallback to Mock Data
- ⚡ Responsive UI
- ☁️ Persistent User Data

## 🚀 Getting Started

### Clone

```bash
git clone https://github.com/yourusername/giga-chad-wallet.git
```

### Install

```bash
npm install
```

### Configure

Create a `.env.local` file using `.env.example`.

### Run

```bash
npm run dev
```
